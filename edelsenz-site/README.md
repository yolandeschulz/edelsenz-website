# EdelSenz website

Static marketing site for **EdelSenz (Pty) Ltd** — *Essential Solutions. Exceptional Results.*

A South African technical ingredient and innovation partner: global & local sourcing,
functional formulation and finished-product manufacturing, idea to shelf.

## Structure
- `index.html` — homepage (overview)
- `division-*.html` — the six division detail pages
- `capabilities.html` — capabilities & technologies
- `assets/` — shared `site.css`, `site.js`, logos, photos and the video montage

Plain HTML/CSS/JS — no build step. Open `index.html` in a browser to view locally.

## Publish free on GitHub Pages
1. Create a new repository on GitHub (e.g. `edelsenz-website`).
2. Upload this whole folder (or push it with git).
3. In the repo: **Settings → Pages → Build and deployment → Source: Deploy from a branch**,
   choose `main` and `/ (root)`, then **Save**.
4. Your live site appears at `https://<your-username>.github.io/<repo>/` within a minute or two.

`.nojekyll` is included so GitHub Pages serves every file as-is.
